# Buisness Page / Portfolio 

## Description

This project is a web application designed with a modern UI/UX, featuring a dark theme, interactive elements, and a parallax-like effect on service cards. It's built using React and Tailwind CSS, showcasing skills in Full Stack Web Development, Blockchain Development, SEO & Digital Marketing, Data Analytics, and Technical Writing.

## Features

- Modern, responsive design with a dark theme.
- Interactive service cards with a parallax-like hover effect.
- Enhanced client links with interactive hover and click effects.
- Comprehensive sections including About, Services, Past Clients, and Contact.
